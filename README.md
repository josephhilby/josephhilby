# Systems & Software Engineer | Former Naval Aviator

I’m a systems-focused engineer who loves getting into the weeds of how things actually work. I spent over a decade as a 
Naval Aviator, where I learned that the best way to conduct high-stakes work is to have a rock-solid grasp of the 
technical fundamentals.  

I’m currently focused on medium-scoped, low-level projects that sharpen my understanding of core CS topics. When I'm away 
from the keyboard, you’ll likely find me working on my homelab setup or getting lost in the classic design philosophy of 
retro games.

## Projects
### Current
- **Hack++:** A full-stack computer architecture project, from logic gates to ISA and custom virtual machine abstractions.
- **FreshlyGround:** A custom high-level programming language and compiler pipeline built on principles of clean, modular code generation.

### Next
- **Xv6:** A Unix Version 6 based instructional OS.
- **Brand Editor:** Make a brand editor from Llama 3 and a styleguide. 

## Technical Stack
- **Languages:** C/C++, Python, TypeScript
- **Systems:** Linux, Docker, GitHub Actions, Cloudflare, Render
