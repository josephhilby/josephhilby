# Technical Program Manager | Systems Engineer | Former Naval Aviator

I’m a systems-focused engineer and Technical Program Manager who loves getting into the weeds of how things actually work. I spent over a decade as a Naval Aviator, where I learned that the best way to manage a high-stakes program is to have a rock-solid grasp of the technical fundamentals.  

I’m currently focused on medium-scoped, low-level projects that sharpen my understanding of core CS topics. When I'm away from the keyboard, you’ll likely find me tweaking my home NAS setup or getting lost in the classic design philosophy of retro games.
